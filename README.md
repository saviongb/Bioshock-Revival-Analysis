# Return to Rapture: A Revival Strategy for BioShock

Data-driven analysis of what players most valued across BioShock 1, 2,
and Infinite — and what 2K should protect in the next entry.

**Status:** In progress (data collection phase)

## Structure
- `notebooks/` — analysis, numbered in reading order
- `src/` — reusable data collection and cleaning code
- `outputs/` — figures and findings
- `data/raw/` is gitignored; fetched via notebook 01 (see methodology)
